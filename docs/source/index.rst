memoryhole
======================================

memoryhole is a library that sends email metadata down the memory hole.

Quickstart guide
-----------------

To protect a message::

  protect(msg)

To unprotect::

  unwrap(msg)

Options
--------

You can pass some options to the public api::

  opts = Options(blah)
